## 🧩 Analyse Technique : Difficulté et Résolution
Dans le cadre de l'analyse du projet **DataNovA** (SaaS de santé), j'ai travaillé sur les problématiques d'interconnexion.

### La difficulté rencontrée
Lors de la mise en place de la communication entre le front-end (React) et l'API (Flask), j'ai été confronté à des erreurs de **CORS (Cross-Origin Resource Sharing)**, bloquant les requêtes de données patients.

### La résolution
J'ai résolu ce problème en configurant le module `flask-cors` sur le serveur Flask pour autoriser explicitement les requêtes provenant du port de développement React. Cela m'a permis d'assurer la fluidité de l'application tout en comprenant les enjeux de sécurité liés aux échanges de données.

## 🧩 Analyse Technique : Difficulté et Résolution
[cite_start]Dans le cadre de l'analyse du projet **DataNovA** (SaaS de santé), j'ai travaillé sur les problématiques d'interconnexion[cite: 26, 28, 33].

### La difficulté rencontrée
[cite_start]Lors de la mise en place de la communication entre le front-end (React) et l'API (Flask), j'ai été confronté à des erreurs de **CORS (Cross-Origin Resource Sharing)**, bloquant les requêtes de données patients[cite: 28, 67].

### La résolution
[cite_start]J'ai résolu ce problème en configurant le module `flask-cors` sur le serveur Flask pour autoriser explicitement les requêtes provenant du port de développement React[cite: 70]. Cela m'a permis d'assurer la fluidité de l'application tout en comprenant les enjeux de sécurité liés aux échanges de données.
